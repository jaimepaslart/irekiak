Audit d'accessibilité WCAG 2.1 AA du projet Irekiak.

Vérifie :
1. Images : alt descriptifs
2. Formulaires : labels associés, erreurs annoncées
3. Navigation : skip links, landmarks ARIA, navigation clavier
4. Contraste : >= 4.5:1 (texte), >= 3:1 (grands textes)
5. Headings : hiérarchie logique
6. Focus : styles visibles, ordre logique
7. Langue : attribut lang correct
8. ARIA : attributs utilisés correctement
9. Animations : prefers-reduced-motion respecté
10. Page accessibilite.vue existante

Lance `npx @axe-core/cli http://localhost:3000 --exit` si le serveur de dev tourne.
Rapport classé par gravité (critique/majeur/mineur).
