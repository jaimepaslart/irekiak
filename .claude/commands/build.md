Lance un build complet et vérifie que tout fonctionne.

1. `pnpm run typecheck` - Vérification TypeScript
2. `pnpm run lint` - ESLint
3. `pnpm run build` - Build production
4. Analyse les erreurs et warnings
5. Propose des corrections si nécessaire

Si des erreurs sont trouvées, corrige-les une par une et relance le build jusqu'à ce qu'il passe.
