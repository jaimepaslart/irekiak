Crée un nouveau composant Vue dans app/components/.

Argument attendu : chemin du composant (ex: gallery/GalleryCard ou shared/HeroSection)

Le composant doit :
1. Utiliser `<script setup lang="ts">`
2. Définir les props avec defineProps typé
3. Définir les events avec defineEmits si nécessaire
4. Utiliser shadcn-vue quand approprié
5. Utiliser useI18n() si du texte affiché
6. Être responsive (mobile-first avec Tailwind)
7. Respecter l'accessibilité : ARIA, rôles sémantiques, alt texts

Crée le fichier dans le sous-dossier approprié de app/components/.
